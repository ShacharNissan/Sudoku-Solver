#############
Sudoku Solver
#############

-----------
Description
-----------
The program can solve the Sudoku puzzel that is given in newspaper, pazzel's books and such,
by going through the lines and boxes and crossing the data of all given information to determine where the next number should be,
till the Sudoku is Solved.

---------
Version 1
---------
Sudoku Solver version 1, By Shachar Nissan, Email: Shachar208@gmail.com

----------
Difficulty
----------
The Sudoku Solver can solve: Easy and Medium level Sudoku Puzzel.

-----
Input
-----
The Sudoku puzzel as shown in the Puzzel.

------
Output
------
The solved Sudoku.
Grey Cells - The given information.
White Cells - The program Solution
